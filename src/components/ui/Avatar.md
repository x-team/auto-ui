Basic usage:

```js
<Avatar
  src={'https://www.gravatar.com/avatar/61a58e425da620d1f4839c6af03ce70f'}
/>
```

With alt text:

```js
<Avatar
  src={'https://www.gravatar.com/avatar/61a58e425da620d1f4839c6af03ce70f'}
  alt={'I am Batman!'}
/>
```

Small size:

```js
<Avatar
  src={'https://www.gravatar.com/avatar/61a58e425da620d1f4839c6af03ce70f'}
  alt={'I am Batman!'}
  size={'small'}
/>
```
