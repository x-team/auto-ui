# Auto UI

[![Build Status](https://secure.travis-ci.org/x-team/auto-ui.png)](http://travis-ci.org/x-team/auto-ui)

_UI component library for the Auto platform._

## Requirements

You need to install:

- [`nvm`](https://github.com/creationix/nvm)

## Get started

- `nvm use v8.4.0`
- `npm install`
- `npm start`
